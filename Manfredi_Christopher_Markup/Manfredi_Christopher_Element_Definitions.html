Definitions For Chapter 2 And 3

The < h1 >
Tag for main headings
< h2,h3,h4,h5,h6>
Tags for sub headings, the smaller the number the bigger the text
The < p >
Tag creates a new paragraph
< b >
Tag to bold text
The < i >
Tag to make text italic
< sup >
Tag to super script code
The < sub >
Tag used when chemical compounds are used
< br / >
Tag used when you want a breaks in your website
The < hr / >
Tag use when you want to add a new chapter or scene
< strong >
Tag used when you want something to stick out
The < em >
Tag used when you want to add emphasis on the word
< blockquote >
Tag used when you want to add a quote but you dont want to show the viewers where you got the quote from
The < q >
Tag used when you want to put quotes inside the paragraphs
< abbr >
Tag used when you want to show the little box that shows the full meaning when hovering over
The < cite >
Tag used when you want to cite a website you got information from
< dfn >
Tag used when you want to define a word in the paragraph.
The < address >
This tag is used when you want to add an address,phone number,email
<ins >
Tag used when you want to show text that has been hidden in the page
The < del >
Tag used when you want to show text that was inserted into the page.
< s >
Tag used when you want to show something that has no value but not delete it
The < ol >
Tag used when you want to show a ordered-list
< li >
Tag used when you want to show list items
The < ul >
Tag used when you want to show an unoredered list
< li >
Tag used when you want to show listed items
The < dl >
Tag used when you have a list of definitions
< dt >
Tag used when you have terms being defined
The < dd >
Tag used to contain definitions
